I4 — π binary expansion (deterministic but no naive structure)
------------------------------------------------------------------------------
  π binary (8192 bytes)                             |x|= 65536b  K_upper= 65720b  rate=1.0028
  Expectation: rate close to 1 under naive compressors —
  π binary is normal under standard tests, so gzip/bz2/lzma
  cannot find structure. K(π) is small in absolute terms but
  COMPUTABLE upper bounds give rate ≈ 1. This shows the gap
  K_upper - K(true): π is the canonical 'looks random under
  computable compression' instance, illustrating that f_rational^alg
  via computable upper bounds is honestly an UPPER bound, not
  the true K(x)/|x|.
  → I4 PASS = True  (rate 1.0028 ≥ 0.9)

  Interpretation: I4 = the framework's analogue of Stark's
  'numerical L-ratio is upper bound on true rationality content'.
  K_upper(π) >> K_true(π), and the gap measures uncomputable
  residue. This is consistent: f_rational > 0 holds for π
  (true K is large in some asymptotic sense), but the
  COMPUTABLE bound is conservative.

V1 — conditional reduction K(x | B) < K(x) when B has structure
------------------------------------------------------------------------------
  Test: x = periodic + 5% noise (= I5),
        B = the bare periodic backbone (= I2)
  Expectation: K(x | B) < K(x), since B carries the structural
  prior that lets the conditional encoder spend bits only on noise.
  K_upper(x)        =   8096 bits
  K_upper(x | B)    =   8072 bits  (joint - K_upper(B))
  reduction         =     24 bits  (0.30%)
  → V1 PASS = True  (K(x | B) < K(x), structural conditioning
    reduces algorithmic content, matching f_rational^alg's L0 A1
    'reducible by better gauge / structural background' property)
