## §2 Hypothesis: physical constraints → paradigm selection

### §2.1 Length-scale → speed → density hierarchy hypothesis

**Hypothesis 2.1 (physical-constraint paradigm selection)**: biological communication paradigm selection is driven by the following physical constraint chain (working hypothesis, this direction paper):

| Physical axis | Constraint | Paradigm requirement | I-series match |
|---|---|---|---|
| **Length** | μm (synapse) ↔ mm (axon) ↔ cm (cortex) ↔ m (sciatic) ↔ km (to be examined: plant root chemical signalling, slime mold gradient sensing, possibly forest-scale mycorrhizal networks) | larger scale → coordination needed | long → P1 phase / short → Shannon binary |
| **Speed** | 1 ms (action potential) ↔ 100 ms (EEG burst) ↔ s (hormone) ↔ min-h (DNA replication) | high speed → simple encoding | fast → Shannon binary / slow → P3 qutrit-style |
| **Error tolerance** | 1 bit error tolerable (axon) ↔ 0 bit error allowed (DNA replication) | low error → redundancy needed | low-error → quantum / DNA-style redundancy / high-error → Shannon lossy |
| **Bandwidth** | nat / sec | high bw → density needed | high-bw → P3 qutrit / low-bw → Hartley scalar |

### §2.2 5-paradigm biological instance speculation

| Biological system | Physical character | I-series paradigm conjecture | Status |
|---|---|---|---|
| **DNA (codon)** | long storage (~Gb), low speed (~kb/s), 0-error tolerance, error-correcting redundancy | **P3 qutrit codebook lift** (codon = 3-letter window) | ESTABLISHED biological instance (Paper F F1, 2026-04-26) |
| **EEG (oscillation)** | long-range (cm-scale cortex), medium speed (10-100 Hz oscillation), phase-coherent broadcast, ~2x density via phase | **P1 σ\*-phase channel** | ESTABLISHED biological instance (sigma_star.md Entry 2, 2026-04-09) |
| **Action potential (axon)** | short-range (mm-cm), fast (1-100 m/s; unmyelinated 0.5-2 m/s vs. myelinated 50-120 m/s), binary spike, 1-bit error tolerable | **Shannon binary BSC/BEC** | classical, signal-theoretic base instance |
| **Synaptic transmission** | very short (μm), fast (~1 ms), continuous-graded vesicle release | **Shannon AWGN (Gaussian channel)** | classical |
| **Hormonal signalling (endocrine)** | very long-range (whole-body), very slow (sec-h), concentration-graded broadcast | **Shannon AWGN slow-channel** | classical |
| **Quorum sensing (bacteria)** | medium range (mm), medium speed (min), threshold-collective | **Hartley + threshold (P1 simplified)** | classical / framework hybrid |

**Observation**: biology selects across the **5-paradigm spectrum** according to the physical constraint hierarchy — signal theory (Shannon binary / AWGN) is short-range, fast, error-tolerant; I-series novel (P1 / P3) is long-range, error-critical, density-limited.
